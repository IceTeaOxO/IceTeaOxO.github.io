使用Hugo+GitHub Page打造自己的Blog

14 Jan 2023, 14:28
hugo
  1. 先下載Hugo

mac的用戶可以使用

brew install hugo

window的用戶請參考官方文件

  1. 在GitHub建立mysite的repository，clone到自己的桌面

  2. 在GitHub建立.github.io的repository，並clone到自己的桌面，並增加README.md後push到GitHub

git clone https://github.com/<User>/<User>.github.io.git
cd <User>.github.io
echo '`Hello World`' > README.md
git add .
git commit -m "init commit"
git push origin main

4.接下來使用Hugo建立自己的網站

cd desktop
hugo new site mysite
  1. 創建新的post並且修改mypost.md的draft為false
cd mysite
hugo new posts/mypost.md

draft: false
  1. 運行Hugo server查看是否成功
hugo server
  1. 選擇自己想要的主題，並且進入到該主題的GitHub並複製clone URL。

切換目錄到themes，並clone該主題的GitHub，或者使用submodule

cd themes
git clone https://github.com/yoshiharuyamashita/blackburn.git
(git submodule add https://github.com/yoshiharuyamashita/blackburn.git)
  1. 修改config.toml，在後面增加theme = “blackburn”(clone主題的名稱)，並且修改baseURL，整份文件目前如下
baseURL = 'https://<User>.github.io/'
languageCode = 'zn-tw'
title = 'My New Hugo Site'
theme = "blackburn"
  1. 連結.github.io，先回到mysite目錄，確認有沒有public資料夾，如果有可以先將它刪除
cd mysite
git submodule add -b main https://github.com/<User>/<User>.github.io.git public
  1. 回到mysite目錄，打包hugo，hugo會自動產生網頁的檔案在public，進入到public push上github，會發現檔案出現在自己的.github.io
hugo
cd public
git add .
git commit -m "commit"
git push origin main
  1. 進入到Setting>Pages>調整Branch為main等待幾分鐘就可以看到自己的網站了

略為匆忙的紀錄，後續會再進行修改
